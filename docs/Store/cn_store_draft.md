# 国内渠道商店页草案

## 定位文案
一款“可爱任务 + 硬核动作”的纸片人 Roguelike。你越熟练，主角就越强，而不是无脑堆数值。

## 特色卖点
- 主打手感：闪避无敌帧、取消窗口、处决反馈。
- 反差叙事：萌系任务目标与紧张战斗节奏并存。
- 高复玩：随机房间与遗物组合，每局策略不同。
- 地区版本适配：合规模式下保留冲击手感。

## 定价建议
- 首发价：42-58 元区间。
- 首周折扣：10%-15%。

## 合规重点
- 使用 CN 资源包（`GAME_REGION=cn`）。
- 禁用高强度血腥表现与处决演出。
- 商店文案规避写实暴力描述。
- 素材来源与授权证明归档齐全。

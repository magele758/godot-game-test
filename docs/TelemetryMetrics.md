# Telemetry Metrics

## 关键事件
- `run_started`：开始新局（含 seed、武器、地区）。
- `run_completed`：结算（胜负、时长、伤害、熟练度）。
- `player_died`：死亡点（房间编号、敌人密度、资源状态）。
- `room_cleared`：房间清空（耗时、是否无伤）。

## 核心指标
- 通关率（按地区、版本、武器分组）。
- 平均单局时长与中位数。
- 首次死亡分布（第几房间最容易崩盘）。
- 完美闪避率与连段多样性。
- 不同武器使用占比与胜率差异。

## 平衡迭代节奏
- 每两周固定一次数据回看。
- 每次最多调整 1-2 个核心变量（敌人攻频、玩家冷却、遗物权重）。
- 调参后必须走完整回归套件，避免手感漂移。

# Content Style Guide

## 目标
在同一套世界观内稳定输出“可爱任务 + 高冲击战斗反馈”的反差体验，并可按地区切换表现强度。

## 叙事语气
- 任务文本采用轻松语气，角色动机偏日常化、童趣化。
- 禁止在任务文本中直接描述写实残忍细节。
- 战斗结束语可夸张，但应保持黑色幽默而非现实仇恨导向。

## 视觉层级
- 第一层（世界）：纸片拼贴、明亮饱和色、圆角卡通轮廓。
- 第二层（战斗）：命中特效、屏幕震动、短时停顿增强打击感。
- 第三层（血腥反馈）：以配置项控制颜色、粒子、残留时长。

## 地区化开关
- `gore_intensity`: `0-2`
  - `0`：无血色，仅星形或纸屑粒子。
  - `1`：低强度暗色粒子与短时地面残留。
  - `2`：完整红色粒子、击杀强化反馈。
- `execution_enabled`:
  - `false`：禁用处决镜头与处决文本。
  - `true`：开启完整处决动作。

## 任务反差模板
1. 可爱目标（例如找回走失玩偶）
2. 战斗冲突（阻拦敌群）
3. 黑色幽默收束（玩偶其实是战利品容器）

## 内容审核清单
- 是否触发地区化禁用词。
- 是否存在对现实群体的明确指向。
- 是否有未记录来源的 AI 素材。

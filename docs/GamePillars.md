# Game Pillars

## 项目定位
- 类型：2D 纸片人风格 Roguelike 动作游戏。
- 核心情绪：可爱叙事与暴力战斗的反差冲击。
- 商业目标：PC 双线发行（海外 Steam + 国内渠道）。

## 三条产品支柱
1. 技术成长优先于数值成长  
   角色强度主要来自玩家操作熟练度（闪避、连段、走位、资源判断），元进度用于解锁打法分支而非线性堆攻防。
2. 血萌反差形成记忆点  
   主线任务、NPC 互动和目标表达使用轻松可爱语气，战斗反馈保持高冲击力，形成稳定反差体验。
3. 短局快反馈高复玩  
   单局目标时长 20-30 分钟，失败后快速回营地，提供明确成长反馈与下一局策略变化。

## 主循环
1. 营地选择武器与目标任务。
2. 进入随机房间连战，获取遗物与临时强化。
3. 通过操作表现（完美闪避、无伤房间、连段多样性）获得熟练度奖励。
4. 击败区域首领或死亡返回营地，结算元进度并解锁新动作分支。

## 成长系统边界
- 允许：
  - 解锁新动作（派生攻击、反击技、位移取消）。
  - 提升战斗可塑性（新连段路径、资源回收机制）。
- 禁止：
  - 直接按游玩时长线性堆高基础伤害。
  - 让新手通过纯数值跳过核心战斗学习。

## 风格与合规边界
- Global 版本：保留完整血腥反馈（血色、击杀效果、处决表现）。
- CN 版本：弱化血腥粒度（颜色替换、粒子数量减少、禁用肢解语义表现）。
- 叙事底线：可爱任务不涉及现实敏感暴力对象，保持虚构世界观。

## 成功指标（前 6 个月）
- 原型阶段：核心战斗 60 FPS 稳定，可持续可玩 10 分钟。
- MVP 阶段：单局可玩闭环完成率 >= 80%（内部测试）。
- 市场阶段：Demo 留存与愿望单增长曲线稳定，形成可复用宣发素材管线。
